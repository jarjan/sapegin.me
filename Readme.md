## Running locally

First clone the repo, install dependencies and build the site:

```bash
git clone https://github.com/sapegin/sapegin.me.git
cd sapegin.me
npm install
npm run build
```

Then run dev server (it will refresh the page after any changes in JS, styles or content):

```bash
npm start
```
